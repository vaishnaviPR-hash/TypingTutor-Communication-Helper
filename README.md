# TypingTutor-Communication-Helper

Welcome to **TypingTutor-Communication-Helper**! This is a fun and interactive web app designed especially for kids to help them communicate and learn using a light-up keyboard and speech features.

## Features

- **Communication Mode:** Type and hear the letters spoken aloud with fun sound effects.
- **Learning Mode:** Guided lessons that highlight keys and encourage kids to learn letters and words interactively.
- **Engaging UI:** Large, colorful buttons with animations that make learning enjoyable.
- **Responsive Design:** Works well on different screen sizes including landscape mode on tablets and phones.
- **Accessible:** Keyboard navigation and ARIA labels to support all users.

## How to Use

1. Open the app in your browser.
2. Use the welcome page to navigate to Learning Mode or Communication Mode.
3. In Learning Mode, select a module to start a guided lesson with on-screen instructions and highlighted keys.
4. In Communication Mode, freely type with the keyboard and hear the letters or sentences read aloud.

## File Structure

- `index.html` — Landing page for app navigation.
- `communicationMode.html` — The main communication mode.
- `learningMode.html` — Learning mode with interactive lessons.
- `m1.html`, `m2.html`, `m3.html` — Individual lesson modules.

## Technologies Used

- HTML, CSS, JavaScript
- Web Speech API for speech synthesis
